Here are some things that are required for this rice:
Kitty: sudo pacman -S kitty
Hyprland: sudo pacman -S hyprland
some qt6 and qt (in general) utilities for quickshell (I can't help with that.)
quickshell: sudo pacman -S quickshell
wallpaper engine and wallpapers: yay -S linux-wallpaperengine-gui after that, go to steam, (if u haven't already), buy wallpaper engine, and download this folder (all the wallpapers I have, including the one shown in the video): https://steamcommunity.com/sharedfiles/filedetails?id=3711807089
** MAKE SURE YOU HAVE GTK INSTALLED **
